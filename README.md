# 📝 BlogNest

A modern blog publishing web application built using **React (Vite)** and **Appwrite**.
BlogNest allows users to create, manage, and explore blog posts with a clean and responsive interface.

---

## 🚀 Tech Stack

* **Frontend:** React + Vite
* **Backend & Services:** Appwrite
* **Authentication:** Appwrite Auth
* **Database:** Appwrite Database
* **Styling:** Tailwind CSS

---

## ✨ Features

* 🔐 User Authentication (Signup / Login)
* 📝 Create, edit, and delete blog posts
* 📄 View all posts in a clean UI
* ⚡ Fast performance with Vite
* ☁️ Backend powered by Appwrite (Database + Auth)
* 📱 Fully responsive design

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/arihantjain-14/BlogNest.git

# Navigate to project folder
cd BlogNest

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file using `.env.sample` as reference:

```env
VITE_APPWRITE_URL=your_appwrite_endpoint
VITE_APPWRITE_PROJECT_ID=your_project_id
```

## 🧠 How It Works

* Appwrite handles authentication and database operations
* React manages UI and state
* Users can create and manage blog posts through an intuitive interface
* Data is fetched and updated via Appwrite services
